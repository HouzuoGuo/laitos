Hello from appengine!

This is a demo app, brought to you by the laitos open source project.

Check out these URLs:

  • /cmd - run a shell command by entering something like "password.s echo 123".
  • /info - inspect the program and system environment.
  • /proxy - a rudimentary web proxy. Write down the intended destination URL like this: "/proxy?u=https://google.com"
  • /myrequest - read back my HTTP request and see how middleware(s) have mangled with and altered your HTTP request.
